# Memory Card Game

A React-based memory card game built with Vite and styled with Tailwind CSS.

## How to Play

- Click on cards to earn points
- Each card can only be clicked once per round
- Clicking a card you've already clicked resets your score to 0
- Try to beat your best score!

## Features

- Responsive design that works on all screen sizes
- Card shuffling after each click
- Score tracking with best score persistence
- Beautiful hover effects and transitions

## Tech Stack

- React
- Vite
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the provided URL.
